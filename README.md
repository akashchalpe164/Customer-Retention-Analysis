# Customer-Retention-Analysis
This project analyzes customer churn behavior to identify key factors affecting customer retention. It helps businesses understand why customers leave and provides data-driven insights to improve retention strategies.

## Objectives
Analyze customer churn rate
Identify high-risk customers
Understand impact of contract type, tenure, and services
Provide actionable business recommendations

## Dateset Used
- <a pref ="https://github.com/akashchalpe164/Customer-Retention-Analysis/blob/main/telco_churn_unclean.csv"> Dataset </a>

## Key Insights
Total Customers: 7043
Churn Rate: 26.54%
Month-to-Month contracts have highest churn
Customers with shorter tenure (0–6 months) churn more
High-risk customers identified based on behavior
